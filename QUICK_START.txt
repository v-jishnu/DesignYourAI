================================================================================
  MCQ INGESTION SYSTEM - QUICK START GUIDE (FREE API)
================================================================================

📚 WHAT THIS DOES:
   Automatically extracts interview MCQs from websites/PDFs
   Classifies them using FREE Gemini AI
   Stores in organized Excel database

💰 COST: $0 (100% FREE with Gemini)

⏱️  SETUP TIME: 5 minutes

================================================================================
  STEP 1: GET FREE API KEY (2 minutes)
================================================================================

   1. Visit: https://makersuite.google.com/app/apikey
   2. Click: "Get API Key"
   3. Copy: Your key (starts with AIza...)

================================================================================
  STEP 2: CONFIGURE (1 minute)
================================================================================

   1. Open .env file:
      notepad .env

   2. Add these lines:
      LLM_PROVIDER=gemini
      GEMINI_API_KEY=AIza...paste-your-key-here...

   3. Save and close

================================================================================
  STEP 3: INSTALL DEPENDENCIES (1 minute)
================================================================================

   pip install google-generativeai openai

   OR reinstall all:
   pip install -r requirements.txt

================================================================================
  STEP 4: TEST (1 minute)
================================================================================

   python test_sample.py

   Expected: ✅ ALL TESTS PASSED!

================================================================================
  STEP 5: USE! (One link at a time)
================================================================================

   Command format:
   python main.py --sources "YOUR_URL_HERE"

   Example:
   python main.py --sources "https://www.geeksforgeeks.org/machine-learning-mcqs/"

   Wait for completion, then process next link!

================================================================================
  WORKFLOW (REPEAT UNTIL 800 MCQs)
================================================================================

   1. Find MCQ source (website or PDF)
      ↓
   2. Run command with that ONE source
      ↓
   3. Wait 1-3 minutes (system processes)
      ↓
   4. Check Excel file (data/knowledge_base/mcq_knowledge_base.xlsx)
      ↓
   5. Get next source and repeat!

================================================================================
  WHAT HAPPENS WHEN YOU RUN:
================================================================================

   STEP 1: Extraction
   → Fetches webpage/PDF
   → Finds all MCQ questions
   → Extracts question + 4 options

   STEP 2: Classification (FREE Gemini AI)
   → Sends to Gemini API
   → AI determines:
     • Category: Conceptual/Mathematical/Application
     • Topic: AI/ML/Data Science/System Design
     • Difficulty: Easy/Medium/Hard

   STEP 3: Deduplication
   → Checks against existing MCQs
   → Removes duplicates
   → Keeps only unique ones

   STEP 4: Storage
   → Saves to Excel file
   → Creates backup
   → Updates count

================================================================================
  TROUBLESHOOTING:
================================================================================

   Problem: "API key not found"
   → Check .env file has GEMINI_API_KEY filled in

   Problem: "No MCQs extracted"
   → Source doesn't have clear MCQ format
   → Try different source

   Problem: "google-generativeai not installed"
   → Run: pip install google-generativeai

   Problem: "Rate limit exceeded"
   → Wait 1 minute and try again
