SETUP (One-time, 5 minutes)
================================================================================

Step 1: Get FREE Gemini API Key
   → Visit: https://makersuite.google.com/app/apikey
   → Click "Get API Key"
   → Copy key (starts with AIza...)

Step 2: Configure
   → Open: .env
   → Add:
     LLM_PROVIDER=gemini
     GEMINI_API_KEY=AIza...your-key...

Step 3: Activate Virtual Environment
   → Run: activate.bat
   → You'll see: (.venv) in your prompt

Step 4: Install Dependencies (isolated)
   → Run: pip install --index-url https://pypi.org/simple -r requirements.txt
   → Wait ~2 minutes

Step 5: Test
   → Run: python test_sample.py
   → Expected: ✅ ALL TESTS PASSED!

================================================================================
DAILY USAGE (Every time you use the system)
================================================================================

Step 1: Activate Virtual Environment
   → Run: activate.bat
   → Look for: (.venv) in prompt

Step 2: Process ONE Link
   → Run: python main.py --sources "https://example.com/mcqs"
   → Wait 1-3 minutes

Step 3: Check Results
   → Open: data\knowledge_base\mcq_knowledge_base.xlsx
   → Verify new MCQs added

Step 4: Process Next Link
   → Run: python main.py --sources "next-url"
   → Repeat until 800 MCQs!

Step 5: Deactivate When Done
   → Run: deactivate

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "google-generativeai not found"
Solution: Run activate.bat first

Problem: "pip install fails with 401 error"
Solution: Use --index-url https://pypi.org/simple

Problem: ".venv not found"
Solution: Run: python -m venv .venv

Problem: "Can't activate in PowerShell"
Solution: Use Command Prompt (cmd) instead
