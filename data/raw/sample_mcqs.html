What is machine learning?

A) A subset of artificial intelligence that learns from data
B) A programming language for data analysis
C) A database management system
D) A web development framework
Correct Answer: A

Which algorithm is used for classification?

A) K-means clustering
B) Decision tree
C) PCA
D) Autoencoder
Correct Answer: B

What does overfitting mean?

A) Model performs poorly on training data
B) Model performs well on training but poorly on test data
C) Model cannot learn from data
D) Model has too few parameters
Correct Answer: B

Which metric is used for regression?

A) Accuracy
B) Precision
C) Mean Squared Error
D) F1 Score
Correct Answer: C

What is gradient descent?

A) An optimization algorithm to minimize loss function
B) A classification algorithm
C) A feature selection technique
D) A data preprocessing method
Correct Answer: A
